* * * create virtual environment
- in terminal (gitbash):
    python -m venv [name of environment]


* * * activate virtual environment
- in terminal type (gitbash):
    source [name of environment]/Scripts/activate


* * * create development environment in gitbash:
(this sets up flask to automatically update environment without having to restart flask)
- in terminal:
    export FLASK_ENV=development

    export FLASK_APP=app.py

- to activate:
    flask run 


* * * version control
- create SSH key (gitbash)
- in terminal:
    cd ~/

    pwd (shows current directory)

    mkdir .ssh (. makes hidden folder)

    ssh-keygen.exe
    enter (for first question to save in current directory)
    enter (for no password)

    cat id_rsa.pub
    - create SSH in github

- version control
- in terminal (gitbash):
    touch .gitignore (this will exclude virtual environment from being pushed to github)

    in gitignore folder:
    .gitignore

    /[name of virtual environment]

-- setting up version control (gitbash) (from codemy.com):

*turn on virtual environment

$ git config --global user.name "[Your Name]"
$ git config --global user.email "[you@youraddress.com]"
$ git config --global push.default matching
$ git config --global alias.co checkout
$ git init

- master or main should appear in prompt
- repository created

- now need to "turn on"
- in terminal (gitbash):
    git add .
    git commit -am '[message what changed]'
